# Daily Lines

Daily Lines is a simple React application for writing short daily notes.

## Features
- Add a new note
- Delete a note
- Search notes while typing
- Add notes by pressing Enter

## Built With
- React
- Vite
- JavaScript
- CSS

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the development server:
   npm run dev

## Purpose
This project was built as a small practice project to learn the basics of React state, events, and list rendering.
